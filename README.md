## DeliveryBairro UserApp 
Projeto desenvolvido em React Native+Expo CLI para gestão de pedidos e entregas com foco no segmento de Delivery em Geral, tais como Bares, Restaurantes, Fastfoods, Distribuidoras de Bebidas, Farmárcias, Drograrias, Mercados de Conveniência, Supermercados, Gás de Cozinha, Água Mineral, etc.

## Atualizando o NodeJs/npm, Expo-CLI e EAS-CLI
https://www.npmjs.com/package/npm-windows-upgrade

npm install -g expo-cli
npm install -g eas-cli

## deployment
<strong>eas build -p android --profile preview</strong>
